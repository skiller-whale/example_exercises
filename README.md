# Welcome to Skiller Whale's Hosted Environment!

This contains a simple Python script, and starts the learner sync service running.

## The Sync Service

You should see a bar in the app now that says `"Sync Active"`.

If you edit the python script in the `exercises` folder, a copy of the file is visible to the coach leading the session (in this demo, it is watching for any changes to `.py` or `.js` files).
